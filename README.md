# Validator

Create a record validator via composable sub-validators

## Installation

### Using Opam

```bash
opam install inquire
```

### Using Esy

```bash
esy add @opam/inquire
```

## Usage

### In OCaml

```ocaml
let () = Validator.greet "World"
```
